Hello [Customer],

You’re invited to the Microsoft and CDC Data Centres Secure Cloud Summit on 23 August in Canberra. This is an education and training event designed to offer in depth Microsoft Azure learning and networking tailored specifically to our new Cloud regions in Canberra – the Azure Australia Central Regions.

These two regions, certified by the ASD for both Dissemination Limiting Markers (DLM) and PROTECTED data, are designed to serve the mission-critical needs of Australian and New Zealand Government, National Critical Infrastructure organisations and their trusted suppliers.

At the event you’ll learn how to:
  • Connect securely to Azure in the Australia Central Regions
  • Secure Protected-level workloads in Azure
  • Manage and extend user identities to Azure
  • Complete the Threat and Risk Assessment for applications in Azure
  • Migrate applications and infrastructure to Azure
We’ll demonstrate threat and risk assessments for workloads in Azure, show you how to implement protection for applications in Azure and how to migrate your organisation’s applications and infrastructure. Hear insights and learnings from your peers which you can implement into your own environment.
