Atelier wordpress theme review

Atelier is a premium Wordpress eCommerce theme developed by SwiftIdeas, a power elite author at ThemeForest.

In this quick review article, I will list down all the features of this wonderful eCommerce theme.

Features:

Supports woocommerce additional variation images plugin
Supports easy digital downloads plugin
Compatible with woocommerce 2.6.4
Theme option to disable hero heading transition
Option to make the shop slider appear only on the main shop page
Updated woocommerce quickview (v3.4.2)

Fourteen high quality demos

Mobile Responsive

High page speed

Three product page design variations

Built-in drag and drop header builder with ten different styles

Lots of animations

Custom icons

Four product page display type

Four product page display type

Drag and drop page builder

Selection of page types each having numerous options

Loaded with shortcodes and 100% translatable

Incredible WooCommerce Support – Atelier not only supports, but makes WooCommerce sites look outstanding. We’ve added options for you to set full width shops, with customisable columns, sidebars, and product display types.

Exceptional Code & Framework Quality – We’ve spent years developing the framework on which Atelier is built upon, and it shows with the incredible performance that you can squeeze out of sites built with it. We’re bringing never before seen speed and functionality to the marketplace, the only theme that will allow you to truly set aside the need for any other theme from this moment.

Truly unlimited styling and customisation – We can safely say that when you use Atelier, you are given the power to create something truly unique, without any difficulty at all. We’ve included full colour customisation, and a plethora of layout customisations to ensure that you can design your site just the way you want it.

Child Theme Compatible & Included – Atelier is Child Theme ready, and we’ve even included one already set up, just install and active and you’re good to go!

Full Width & Boxed Layout Theme – Both are selectable with a single click, and you can also control the width of the site at it’s largest viewport, for ultimate control!

Iconmind Icon Font – We’ve taken the expense to include the stunning Iconmind icon font with Atelier, free to use anywhere within the theme. We’ve also included Font Awesome, so you’ll never need to seek out any other icon font.

Multi-site Ready – Atelier has been thoroughly tested on WordPress Multisite installations, with absolutely no issues or limitations.

Gravity Forms Support – Gravity Forms has been tested & styled to match the theme styling – ready for you to create unlimited forms and display them perfectly. We’ve even created a page builder asset specifically for them, so you can quickly access them, and add them anywhere within your page.
